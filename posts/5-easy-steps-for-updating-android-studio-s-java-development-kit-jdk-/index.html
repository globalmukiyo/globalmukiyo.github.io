2. Integrate JDK on Android Studio#

Updating process of JDK in Android Studio is a simple process which can be done by following few basic steps. First of all, open your Android Studio and go to ‘File -> Project Structure’. There, you’ll find a tab labeled as ‘SDK Location’. This is where you can set the Path for JDK. Make sure the path is set to the JDK directory. After that, press ‘OK’ to apply the changes. Now, click on the ‘Sync Project with Gradle Files’ to get the latest version of JDK. Finally, restart your Android Studio and you should now be able to work on your project with the newest version of JDK.
